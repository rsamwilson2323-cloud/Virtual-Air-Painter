# 🎨✨ Virtual Air Painter

Virtual Air Painter is a real-time hand gesture-based drawing app built using **Python 🐍, OpenCV 🎥, and MediaPipe 🧠**.  
Draw in the air using your fingers without touching anything — fully interactive and fun! 🚀

---

## 🚀🔥 Features

✨ Draw in the air using hand gestures  
🎨 Change colors using finger count  
💾 Auto-save drawing using palm close gesture  
🖥️ Full-screen immersive experience  
📁 Saved images stored automatically  

---

## 🧠⚙️ Technologies Used

- Python 🐍  
- OpenCV 🎥  
- MediaPipe 🧠  
- NumPy 🔢

## ✋🎮 Controls & Gestures

### 🎨 Color & Drawing Mode

☝️ 1 Finger  → ❌ No Drawing  
✌️ 2 Fingers → ⚪ White  
🤟 3 Fingers → 🔴 Red  
🖖 4 Fingers → 🔵 Blue  
🖐️ 5 Fingers → 🟢 Green  

---

### 💾🧾 Save & Clear

✊ Close Palm (0 fingers):  
✔️ Saves your drawing  
✔️ Clears the screen  

📁 Saved in:
paint/paint_<timestamp>.bmp  

Example:  
paint/paint_1712345678.bmp  

---

### ❌🚪 Exit

Press **ENTER key ⏎** to exit  

---

## 🖼️⚡ How It Works

🎥 Camera captures hand  
🧠 MediaPipe detects landmarks  
✋ Finger count selects color/mode  
👉 Index finger draws  
🖥️ Canvas overlays on live video  

---

## 📌💡 Notes

💡 Use good lighting  
💡 Keep hand visible  
💡 Move smoothly for best drawing
